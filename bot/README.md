## 📁 ساختار پروژه

```
bot/
├── main.py                 # فایل اصلی ربات
├── api.py                  # سرور API Flask
├── config.py               # تنظیمات و متن‌ها
├── requirements.txt        # وابستگی‌های Python
├── modules/
│   ├── __init__.py
│   ├── database.py        # مدیریت دیتابیس JSON
│   ├── user_manager.py    # مدیریت کاربران و شناسه‌ها
│   └── handlers.py        # Handlers تلگرام
├── database/
│   └── users.json         # ذخیره داده‌های کاربران
└── logs/
    └── bot.log            # لاگ‌های ربات
```

## 🚀 نصب و اجرا

### 1️⃣ نصب وابستگی‌ها
```bash
cd bot
pip install -r requirements.txt
```

### 2️⃣ اجرای ربات
```bash
# Terminal 1
python main.py
```

### 3️⃣ اجرای سرور API (در Terminal جداگانه)
```bash
# Terminal 2
python api.py
```

### 4️⃣ اجرای وب‌سایت
- صفحه HTML را در مرورگر باز کنید

## 🔌 جریان کار

### 1. کاربر /start در تلگرام می‌زند
- ربات شناسه یکتا می‌سازد (مثل: `a1b2c3d4`)
- Chat ID خودکار ذخیره می‌شود
- کاربر پیام خوش‌آمد دریافت می‌کند

### 2. کاربر "📝 ثبت اطلاعات" را کلیک می‌کند
- Web App باز می‌شود
- شناسه یکتا به URL منتقل می‌شود
- فرم نمایش داده می‌شود

### 3. کاربر فرم را پر می‌کند
- اطلاعات به سرور ارسال می‌شود
- داده‌ها برای کاربر جداگانه ذخیره می‌شوند
- پیام تایید نمایش داده می‌شود

### 4. داده‌ها ذخیره شده اند
```json
{
  "a1b2c3d4": {
    "chat_id": 123456789,
    "user_id": "a1b2c3d4",
    "first_name": "علی",
    "last_name": "محمدی",
    "phone": "09121234567",
    "smoker": "no",
    "diabetes": "no",
    "created_at": "2026-04-15T10:30:00",
    "form_updated_at": "2026-04-15T10:35:00"
  }
}
```

## 📱 API Endpoints

### دریافت اطلاعات کاربر
```
GET /api/user/{user_id}
```
**Response:**
```json
{
  "status": "success",
  "user": {...}
}
```

### ثبت داده‌های فرم
```
POST /api/form/submit
Content-Type: application/json

{
  "user_id": "a1b2c3d4",
  "firstName": "علی",
  "lastName": "محمدی",
  "phone": "09121234567",
  "smoker": "no",
  "diabetes": "no"
}
```

### تایید وجود کاربر
```
GET /api/validate-user/{user_id}
```

### دریافت تمام کاربران
```
GET /api/all-users
```
